### 4.3. Cài đặt - cấu hình keepakive

**Cấu hình trên 2 node web server**

- Cài đặt

```
yum install keepalived -y
```

```
cp /etc/keepalived/keepalived.conf /etc/keepalived/keepalived.conf.bak
```

```
systemctl start keepalived
systemctl enable keepalived
```

- Thực hiện trên node master

Chỉnh sửa file `/etc/keepalived/keepalived.conf`

```
vrrp_script chk_httpd {
    script "killall -0 httpd"
    interval 2
    weight 2
}
vrrp_instance VI_1 {
    interface eth0
    state MASTER
    virtual_router_id 51
    priority 101
    virtual_ipaddress {
        103.101.160.25/24
    }
    track_script {
        chk_httpd
    }
}
```

```
systemctl restart keepalived
```

-Thực hiện trên node slave

Chỉnh sửa file `/etc/keepalived/keepalived.conf`

```
vrrp_script chk_httpd {
    script "killall -0 httpd"
    interval 2
    weight 2
}
vrrp_instance VI_1 {
    interface eth0
    state BACKUP
    virtual_router_id 51
    priority 99
    virtual_ipaddress {
        103.101.160.25/24
    }
    track_script {
        chk_httpd
    }
}
```

```
systemctl restart keepalived
```

Kiểm tra node `master` sẽ thấy `IP VIP 103.101.160.25` do keepalive quản lý.
